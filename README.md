# Data-Visualization

**python version 2.7**
## Installing matplotlib
```
$ sudo apt-get install python-matplotlib
```
## Installing Pygal
```
$ pip install --user pygal
$ pip install pygal_maps_world
```
## Installing Requests
```
$ pip install --user requests
```
